# Cybersecurity-audit-olaoyedamilola
##Project Title:Personal Device Audit

## Objective:The aim of this project is to identify the current state of my smart phone device and practical solution to strengthen if any weakness
## Scope:I audited my smart phone device
## Tools Used:
    •Security setting pages
    •System Update Page
    •Device settings
    •Browser security settings
  Files in the Repository
  •checklist.md
  •findings-report.md
  •recommendations.md
  •screenshots
    
Key Findings
	∙	Strong passcode and Face ID were enabled
	∙	iOS was fully updated
	∙	Two-factor authentication was already active
	∙	Some unused applications were still installed
	∙	App permissions had not been fully reviewed
	∙	Bluetooth and Wi-Fi were set to auto-connect to unknown networks
	∙	A password manager was not in use
    
Conclusion
This project helped me understand how to identify common cybersecurity risks on an iOS smartphone and recommend practical solutions to improve personal device security.
