# Smartphone Security Checklist

| Security Check            | Status (Yes/No) | Notes |
|--------------------------|----------------|-------|
| Screen lock enabled      | Yes            | Using fingerprint and PIN |
| Phone updated            | Yes            | Updates sometimes delayed |
| Unknown sources disabled | Yes            | Safer app installation |
| App permissions reviewed | No             | Some apps have too much access |
| 2FA enabled              | Yes            | Enabled on important apps |
| Find My Device enabled   | Yes            | Can track phone if lost |
| Safe Wi-Fi usage         | No             | Sometimes uses public Wi-Fi |
